¿En qué podemos ayudarte?

En Abirtone contamos con los mejores expertos en el diseño y creación de herramientas de seguridad perimetral, análisis y hacking ético.

Experiencia

Nuestros especialistas son autores (o co-autores) de numerosas herramientas hacking y seguridad ámpliamente conocidas:

GoLismero

Herramienta y framework para pruebas de seguridad web que automatiza el lanzamiento de otras herramientas (nmap, OpenVAS, SQLMap, Wfuzz...) y unfica los resultados.

http://www.golismero.com →
Plecost

El analizador de vulnerabilidad para Wordpress y sus plugins más rápido que existe. Capaz de comprobar más de 1000 vulnerabilidades en menos de 1 minuto.

https://github.com/iniqua/plecost →
Topera

Topera es una herramienta que realiza ataques IPv6 que no son detectados por el sistema de detección de intrusos Snort, o sistemas basados en Snort.

http://toperaproject.github.io/topera/ →
ktcal2

Librería y herramienta capaz de realizar ataques de fuerza bruta de contraseñaOs contra servicios SSH, utilizando las funciones de conexión de red de alto rendimiento AsyncIO de Python 3.4.

https://github.com/cr0hn/ktcal2 →
Gasón

El primer plugin para BurpSuite que permite integrar SQLMap con la popular herramienta de hacking.

https://code.google.com/p/gason/ →
OpenVAS for Python

Librería (no oficial) más usada para manejar, conectar y gestionar el escaner de vulnerabilidades, OpenVAS, usando Python.

https://github.com/golismero/openvas_lib →
OMSTD

OMSTD (Open Methodology for Security Tool Developers) es una metodología a modo de guía, con serie de casos de estudio, con los que lograr desarrollar herramientas bien construidas.

http://omstd.readthedocs.org/ →
STB (Security Tools Builder)

Creador de esqueletos de herramientas de seguridad para automatizar la construcción de herramientas de hacking, usando la metodlogía OMSTD.

https://github.com/abirtone/stb →
Info2CPE

Analizador de banners de servidores web capaz de obtener su correspondiente identificador CPE. La biblioteca sigue las especificaciones v2.3 CPE.

https://github.com/cr0hn/info2cpe →
BAZIN

Aistente para desplegar sistemas FreeBSD configurados y securizados con nivel militar.

https://github.com/cr0hn/bazin →
UnderFucking

Aplicación web vulnerable y auto-mutable (capaz de generar contenido de forma dinámica) con el fin de servir de mesa de pruebas para herramientas de hacking.

https://github.com/cr0hn/UnderFucking →
