¿ Qué es AngularJS ?

AngularJS nos ypermitirá hacer interfaces web de manera extremadamente rápida, mantenibles y testeables. AngujarJS cuenta con el respaldo de Google, creador del mismo. Según la web de proyecto:

Angularjs le permite escribir aplicaciones web del lado del cliente como si tuvieras un navegador más inteligente [...] sincroniza automáticamente los datos de la interfaz de usuario [...] para ayudar a estructurar su aplicación mejor y que sea más fácil probar ...

Fuente: Angujarjs.com
