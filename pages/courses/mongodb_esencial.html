¿ Qué es MongoDB ?

MongoDB es un sistema de base de datos, de la nueva familia NoSQL u orientado a documentos, Open Source y gratuita.

MongoDB es una base de datos ágil que permite a los esquemas cambiar rápidamente cuando las aplicaciones evolucionan. Ha sido creado para brindar escalabilidad, rendimiento y gran disponibilidad.
