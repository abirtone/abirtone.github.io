Curso avanzado de programación en Python en el que se tocarán temas como:

  • Mantenibilidad en proyectos Python.
  • Generación automática de plantillas para proyectos.
  • Concurrencia avanzada con hilos, procesos y librerías asíncronas.
  • Estudio del GIL de Python: Problemas, motivos y posibles soluciones.
  • Monkey patching o parcheo al vuelo de librerías.
  • Rendimiento avanzado y compilación de código Python con Cython
  • Estudio del número módulo de Python Asyncio y corrutinas.
  • Modelos de comunicación asíncrona y servicios de mensajería con brokers y ZeroMQ: Sistema de sockets de alto rendimiento para sistema distribuidos e hiperscalables.

Información del curso e inscripción
